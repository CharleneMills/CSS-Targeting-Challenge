> This is a nice looking page but as you can see there are a few styling issues we have going on. The goal of this challenge is to fix these styling erros by ONLY touching the css. No changing the HTML. I have set up the HTML to give you everything you need to 'target' the correct HTML items within the CSS.

> Open the index.html in your code editor

> Open the challenge.css in your code editor

> WITHOUT changing the HTML in the index file...

  • Objective: Target a single unique class and drilling down to the desired element. Change the button in THIS card below to have a greenish background instead of blue. Use #00bc8c. (hint look for a unique class that only this card has.
  • Objective: Target a single unique class and then an element inside that class. In the navigation above. Give all the links, EXCEPT the "Dry CSS Challenge" link, some padding.
  • Objective: Target a class that belongs to all the elements you want to change. In the three cards below, make their text 'seeable' by making the font-color black
  • Objective: Target a class that belongs to all the elements you want to change. Give the three card's below some padding (20px is pretty standard)
  • Black Diamond: Change the third card's button's background color to dark red. (hint you must target more than one class)
